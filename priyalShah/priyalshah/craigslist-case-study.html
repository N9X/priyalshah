REDESIGN

Craigslist

Craigslist is one the oldest and most used website for classified ads about jobs, sales, events, cars everything.

Overview

Craigslist is one the oldest and most used website for classified ads about jobs, sales, events, cars everything. Although being one of the oldest the company has never changed the User interface of the website. When you see something like that, you want to take matters in your hands. That's exactly what I did. For 3 weeks, my partner Garvit and I worked together to improve the UI, Visual design, and interactions.

3 Weeks

Define

Scope

The main objective was to do a complete makeover of Craigslist. We addressed several usability issues and rectify them wit UI and visual design. Our scope was narrowed to the UI, Visual design, and increasing the accessibility due to short-time constraint. We deliberately did not touch upon branding, business strategy, user research, core features and functionality, information architecture, and usability testing.

Problem Definition

Before setting on the mission to redesign Craigslist, we had to determine what the heck is wrong with it in the first place. We started by going through all the pages, navigating through it and listing down all the problems we find. We discovered that there are numerous usability and user experience issues. There are various heuristic violations that plagued the navigation, interaction design, and UI.

Define

Element 1: Visibility

  1. Most people navigate in Craigslist in search for things. Features like search bar is placed along the side-bar making it harder to find and lowering the utility of the website as a whole. The principle of visibility recommends that the product functions be obvious and discoverable.
  2. 'Create a posting' looks like just any other link in the page. The visibility for that button is very low. As a functional CTA, it should be highlighted.
  3. Craigslist also lacks visibility spectacularly mainly due to lack of visual hierarchy. There is very little variation in size, shape, color, and prominence of the elements to instruct users about where and how they should begin. Most elements on the page seems to have same or similar visual importance.

Element 2: Consistency

Craigslist's initial screen suffers from a lack of consistency in its headers. Depending on the page you're on in the site, the location and visual elements are all different and inconsistent for -

  1. How to create a posting
  2. How to get to your account
  3. . How to create an account
  4. Site navigation

Proposed Solution

By providing a single standard header across the entire site with the same aesthetic elements in the same places, the redesign promotes consistency. This will improve user memorability when they want to use basic functions at any page in the site.

This will improve user memorability when they want to use basic functions at any page in the site.

We designed a common header for all the pages. Along with header, we also decided to keep the side-bar fixed. This will allow users to navigate more easily throughout the site without going back to the first page every time they want to select some other category.

Element 3: Affordance

The calendar at the side-bar has a lot of affordance issues. When the mouse hovers on previous dates, the gray color covers a date box just like any other coming-up dates. But previous dates are not clickable while coming-up dates are clickable. This gray covered color gives confusion that users expect next happening as going to a post page of the date.

Some icons used for interacting with listings do not match up with the action they perform. For instance, when browsing listings a “trash can” icon appears after a listing’s title. Commonly, a trash can is used to delete items that the user has control over, but in this context it represents hiding a listing from view. Additionally, there is no appearing text when the cursor hovers over the icon to explicitly indicate what the icon does. The icon does have an explicit label, but only on the listing page, which means the user has to click on the listing before seeing what the button does

Element 4: No Filter or Sorting option

There is no way for the user to filter out or sort the categories. It gets frustrating to look through the entire list

I have added the 'filter' and 'sort' option at the top right corner. When implemented, filters and sorting will not only reduce thousands of products into a manageable amount, but they will also order products in a way that meets the user’s interests and needs. Which in turn, will increase the likeliness of the user finding what they are looking for.

All the options at the left sidebar are shifted under the drop down 'Filter' button.

What your buttons say is as important as how they look. Using the wrong words on your button labels cause users confusion, more work, and slower task times. I have changed the button title from 'update search' to 'apply' as it reduces the confusion and makes it easy for the users to understand what will the reaction be.
